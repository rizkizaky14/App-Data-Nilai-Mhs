1. pilih folder yang ingin dijadikan repo lokal 2. git init dalam folder yang ingin dijadikan repo lokal 3. cek terminal : git config user.name 4. cek remoterepo yang di tuju : get remote -v 5. jika tidak ada tambahkan remote repo : gitremote add origin [url .git] 6. cek git remote -v lagi note : cara update git 7. git add [nama file], akan tetapi jika ingin menambahkan semua file terbaru gunakan git add . 8. git commite -m "Message": menaikan file menjadi commite dengan Message 9. git push origin master : origin disesuaikan dengan nama remote server dan master adalah branch yang saat ini digunakan
